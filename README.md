# Spring Boot製ゲーム管理アプリ

TODOアプリの変形である。学習用

---

## 主な機能

- ユーザー認証（ログイン／ログアウト）
- ゲームの登録（タイトル、ジャンル、進行状況）
- ゲーム一覧表示
- キーワード検索（タイトル／ジャンル）
- ゲーム情報の編集・削除

---

## 使用技術

| 分類 | 技術 |
|--------|----------------|
| バックエンド | Spring Boot |
| フロント | Thymeleaf, HTML, CSS |
| データベース | H2 Database |
| ビルド管理 | Maven |
